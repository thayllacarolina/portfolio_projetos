Projetos

Desenvolvimento de dashboard para negócio usando Streamlit

Nesse projeto, as habilidades de análise de dados utilizando a linguagem de programação Python, o pensamento estratégico sobre o modelo de negócio, junto com o uso do framework de desenvolvimento web Streamlit, foram usados para desenvolver um dashboard com as principais métricas e análise descritiva de uma empresa marketplace de restaurantes na India.
O resultado final desse projeto foi um dashboard hospedado em um ambiente Cloud, o qual foi disponibilizado em um link web.

As ferramentas utilizadas foram:
  • Python (Pandas, Plotly, Folium)
  • Jupyter Lab
  • Terminal
  • Streamlit e Streamlit Cloud
  • Github
  • SAIBA MAIS
